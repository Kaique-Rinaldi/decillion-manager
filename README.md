# Decillion Manager

Sistema de gestão de clientes desenvolvido em Portugol, com foco em organização de projetos, controle financeiro e escalabilidade para um futuro SaaS.

---

## Funcionalidades

- Cadastro de clientes
- Login administrativo
- Busca de clientes
- Alteração de dados
- Exclusão de clientes
- Controle de status de pagamento
- Gestão de status de projetos
- Relatórios financeiros (valores a receber e recebidos)
- Filtros de clientes (pagos e pendentes)

---

## Objetivo

Este projeto foi desenvolvido com o objetivo de evoluir para um sistema real de gestão (SaaS), voltado para:

- Freelancers
- Agências
- Pequenas e médias empresas

---

## Tecnologias utilizadas

- Portugol
- Lógica de Programação
- Estruturas de decisão e repetição
- Manipulação de dados em memória
